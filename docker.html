Projet Docker – Architecture Fullstack

Docker · Docker Compose · Angular · Flask · PostgreSQL

Présentation du projet

Dans le cadre de ce projet, j’ai conçu et déployé une architecture fullstack entièrement conteneurisée à l’aide de Docker.

L’objectif était de proposer une solution simple à déployer, scalable et intégrant une base de données sécurisée, prête à être utilisée sur n’importe quelle machine hôte.

Architecture & orchestration

  • Architecture multi-services : frontend, backend et base de données
  • Services isolés et indépendants
  • Orchestration complète via Docker Compose
  • Déploiement automatisé de l’environnement
  • Scalabilité facilitée des services

Sécurité & persistance des données

La base de données PostgreSQL bénéficie d’une sécurisation renforcée grâce à l’utilisation de Docker Secrets.

Le mot de passe de connexion est stocké dans un fichier séparé du code source, évitant toute exposition dans les variables d’environnement ou dans le dépôt Git. La persistance des données est assurée via des volumes Docker.
